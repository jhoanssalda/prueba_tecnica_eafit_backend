# prueba_tecnica_eafit_backend
Repositorio que contiene el backend de la prueba técnica enviada por EAFIT y SIATA, realizado con NodeJS, que expone APIs para CRUD de envíos marítimos y envíos terrestres

#Instalación

Previamente, se debe haber instalado NodeJS, si no se ha instalado, se deja link de descarga de la última versión disponible:
https://nodejs.org/en/download

Clonar el repositorio con el comando:
git clone https://github.com/jhoanssalda/prueba_tecnica_eafit_backend.git

Después de la clonación, ir a la carpeta del repositorio clonado. Si se trabaja con Visual Studio Code, solo falta abrir una terminal en la carpeta y escribir:
code .

Desde allí, abrir una nueva terminal, y ecribir:
npm install

Con esto, se podrán descargar las dependencias necesarias para correr el proyecto.

Luego de instaladas las dependencias, escribir en la terminal:
npm run dev

Se recomienda la instalación de XAMPP para el manejo de la base de datos con MySQL.
